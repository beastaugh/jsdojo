Constraint 3

Contrary to constraint 1, some duplicate items should appear to the user. These are updates to previous items.

They have an update flag set on the object, e.g. 

{
    "text":      "Text of the news item...",
    "timestamp": 1279192986,
    "id":        "1377294734da555d72ad407cac9e4b83e1cba172",
    "update":    true
}

Updates should be inserted into the document.
