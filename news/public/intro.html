Introduction

The basic problem you have to solve is: write a frontend for a live news service. It should display each news item to the user as it arrives.

A server has been set up to provide the news feed. You can access the data here:

http://beastaugh.othermedia.com:4567/news.json

The template page contains three pieces of JavaScript to get you started: the NewsService class and two helper functions, makeNewsItemHTML and insertNewsItem.

http://beastaugh.othermedia.com:4567/template.html

View source and copy the contents into a local file.

The basic idea: create a new instance of the NewsService class and use the on method to listen to the data event. A skeleton is provided on the template page.

The news service provides a list of zero or more news items. You need to insert these into the page as they arrive. Call the makeNewsItemHTML helper function with one argument (a news item object) to create a DOM representation of the news item.

You can use the insertNewsItem helper function to insert that DOM element into the document. Pass in another DOM element as a second argument to insert the news item after that element.

News items have the following structure: 

{
    "text":      "Text of the news item...",
    "timestamp": 1279192986,
    "id":        "1377294734da555d72ad407cac9e4b83e1cba172"
}

There are three additional constraints. You have ten minutes to meet each constraint, after which the next constraint will be provided.
